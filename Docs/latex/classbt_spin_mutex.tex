\doxysubsection{Detailed Description}
\doxylink{classbt_spin_mutex}{bt\+Spin\+Mutex} -- lightweight spin-\/mutex implemented with atomic ops, never puts a thread to sleep because it is designed to be used with a task scheduler which has one thread per core and the threads don\textquotesingle{}t sleep until they run out of tasks. Not good for general purpose use.
