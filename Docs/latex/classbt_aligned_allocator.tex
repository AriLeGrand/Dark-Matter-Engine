class bt\+Aligned\+Allocator$<$ T, Alignment $>$}
The \doxylink{classbt_aligned_allocator}{bt\+Aligned\+Allocator} is a portable class for aligned memory allocations. Default implementations for unaligned and aligned allocations can be overridden by a custom allocator using bt\+Aligned\+Alloc\+Set\+Custom and bt\+Aligned\+Alloc\+Set\+Custom\+Aligned.
