\doxysubsection{Detailed Description}
Minkowski\+Penetration\+Depth\+Solver implements bruteforce penetration depth estimation. Implementation is based on sampling the depth using support mapping, and using \doxylink{struct_g_j_k}{GJK} step to get the witness points.
