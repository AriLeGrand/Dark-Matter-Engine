\doxysubsection{Detailed Description}
The \doxylink{class_g_i_m___c_o_n_t_a_c_t}{GIM\+\_\+\+CONTACT} is an internal GIMPACT structure, similar to \doxylink{classbt_manifold_point}{bt\+Manifold\+Point}. \begin{DoxyRefDesc}{Todo}
\item[\mbox{\hyperlink{todo__todo000025}{Todo}}]\+: remove and replace \doxylink{class_g_i_m___c_o_n_t_a_c_t}{GIM\+\_\+\+CONTACT} by \doxylink{classbt_manifold_point}{bt\+Manifold\+Point}. \end{DoxyRefDesc}


Structure for collision results Functions for managing and sorting contacts resulting from a collision query. Contact lists must be create by calling GIM\+\_\+\+CREATE\+\_\+\+CONTACT\+\_\+\+LIST After querys, contact lists must be destroy by calling GIM\+\_\+\+DYNARRAY\+\_\+\+DESTROY Contacts can be merge for avoid duplicate results by calling gim\+\_\+merge\+\_\+contacts
