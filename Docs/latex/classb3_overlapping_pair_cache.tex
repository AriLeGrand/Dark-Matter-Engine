\doxysubsection{Detailed Description}
The \doxylink{classb3_overlapping_pair_cache}{b3\+Overlapping\+Pair\+Cache} provides an interface for overlapping pair management (add, remove, storage), used by the b3\+Broadphase\+Interface broadphases.
