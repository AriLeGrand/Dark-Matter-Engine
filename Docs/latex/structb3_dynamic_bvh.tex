\doxysubsection{Detailed Description}
The \doxylink{structb3_dynamic_bvh}{b3\+Dynamic\+Bvh} class implements a fast dynamic bounding volume tree based on axis aligned bounding boxes (aabb tree). This \doxylink{structb3_dynamic_bvh}{b3\+Dynamic\+Bvh} is used for soft body collision detection and for the \doxylink{structb3_dynamic_bvh_broadphase}{b3\+Dynamic\+Bvh\+Broadphase}. It has a fast insert, remove and update of nodes. Unlike the b3\+Quantized\+Bvh, nodes can be dynamically moved around, which allows for change in topology of the underlying data structure.
