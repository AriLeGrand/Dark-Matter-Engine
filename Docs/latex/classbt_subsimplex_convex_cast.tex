Simsimplex\+Convex\+Cast calculate\+Time\+Of\+Impact calculates the time of impact+normal for the linear cast (sweep) between two moving objects. Precondition is that objects should not penetration/overlap at the start from the interval. Overlap can be tested using \doxylink{classbt_gjk_pair_detector}{bt\+Gjk\+Pair\+Detector}. take relative motion

Just like regular \doxylink{struct_g_j_k}{GJK} only add the vertex if it isn\textquotesingle{}t already (close) to current vertex, it would lead to divisions by zero and NaN etc.
