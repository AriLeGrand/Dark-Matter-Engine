\doxysubsection{Detailed Description}
The \doxylink{classbt_dispatcher}{bt\+Dispatcher} interface class can be used in combination with broadphase to dispatch calculations for overlapping pairs. For example for pairwise collision detection, calculating contact points stored in bt\+Persistent\+Manifold or user callbacks (game logic).
