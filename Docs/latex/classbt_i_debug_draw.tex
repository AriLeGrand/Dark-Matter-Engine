\doxysubsection{Detailed Description}
The \doxylink{classbt_i_debug_draw}{bt\+IDebug\+Draw} interface class allows hooking up a debug renderer to visually debug simulations. Typical use case\+: create a debug drawer object, and assign it to a \doxylink{classbt_collision_world}{bt\+Collision\+World} or \doxylink{classbt_dynamics_world}{bt\+Dynamics\+World} using set\+Debug\+Drawer and call debug\+Draw\+World. A class that implements the \doxylink{classbt_i_debug_draw}{bt\+IDebug\+Draw} interface will need to provide non-\/empty implementations of the the draw\+Line and get\+Debug\+Mode methods at a minimum. For color arguments the X,Y,Z components refer to Red, Green and Blue each in the range \mbox{[}0..1\mbox{]}
