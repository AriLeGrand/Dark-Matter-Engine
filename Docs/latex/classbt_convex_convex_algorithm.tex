\doxysubsection{Detailed Description}
Enabling USE\+\_\+\+SEPDISTANCE\+\_\+\+UTIL2 requires 100\% reliable distance computation. However, when using large size ratios \doxylink{struct_g_j_k}{GJK} can be imprecise so the distance is not conservative. In that case, enabling this USE\+\_\+\+SEPDISTANCE\+\_\+\+UTIL2 would result in failing/missing collisions. Either improve \doxylink{struct_g_j_k}{GJK} for large size ratios (testing a 100 units versus a 0.\+1 unit object) or only enable the util for certain pairs that have a small size ratio The convex\+Convex\+Algorithm collision algorithm implements time of impact, convex closest points and penetration depth calculations between two convex objects. Multiple contact points are calculated by perturbing the orientation of the smallest object orthogonal to the separating normal. This idea was described by Gino van den Bergen in this forum topic \href{http://www.bulletphysics.com/Bullet/phpBB3/viewtopic.php?f=4&t=288&p=888\#p888}{\texttt{ http\+://www.\+bulletphysics.\+com/\+Bullet/php\+BB3/viewtopic.\+php?f=4\&t=288\&p=888\#p888}}
