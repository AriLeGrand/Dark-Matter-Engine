\doxysubsection{Detailed Description}
The \doxylink{classb3_triangle_callback}{b3\+Triangle\+Callback} provides a callback for each overlapping triangle when calling process\+All\+Triangles. This callback is called by process\+All\+Triangles for all b3\+Concave\+Shape derived class, such as b3\+Bvh\+Triangle\+Mesh\+Shape, b3\+Static\+Plane\+Shape and b3\+Heightfield\+Terrain\+Shape.
