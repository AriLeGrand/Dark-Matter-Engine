\doxysubsection{Detailed Description}
The \doxylink{class_hull_library}{Hull\+Library} class can create a convex hull from a collection of vertices, using the Compute\+Hull method. The bt\+Shape\+Hull class uses this \doxylink{class_hull_library}{Hull\+Library} to create a approximate convex mesh given a general (non-\/polyhedral) convex shape.
