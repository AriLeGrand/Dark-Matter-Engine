class bt\+Axis\+Sweep3\+Internal$<$ BP\+\_\+\+FP\+\_\+\+INT\+\_\+\+TYPE $>$}
The internal templace class \doxylink{classbt_axis_sweep3_internal}{bt\+Axis\+Sweep3\+Internal} implements the sweep and prune broadphase. It uses quantized integers to represent the begin and end points for each of the 3 axis.
