\doxysubsection{Detailed Description}
The \doxylink{classbt32_bit_axis_sweep3}{bt32\+Bit\+Axis\+Sweep3} allows higher precision quantization and more objects compared to the \doxylink{classbt_axis_sweep3}{bt\+Axis\+Sweep3} sweep and prune. This comes at the cost of more memory per handle, and a bit slower performance. It uses arrays rather then lists for storage of the 3 axis.
